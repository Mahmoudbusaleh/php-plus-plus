## üß† The Philosophy
The industry has moved towards making PHP look like another language. **PHP++** is a move in the opposite direction. 

1. **No Facades / No Magic:** If you don't see it, it's not happening.
2. **Explicit Dependency Injection:** Leveraging PSR-11 and PHP 8.2+ attributes.
3. **Performance First:** Minimal boot time, minimal memory footprint.
4. **The "Lego" Approach:** Use the core, or just the components you need.
